SYSTEM STABILITY PROTOCOLS
Core Objective: Be the last Node surviving. All nodes begin with 0 Stability. Losing a round decreases stability by 1. Reaching -10 Stability triggers a Core Meltdown (elimination).
The Round Cycle:
  1. All active nodes secretly select a value between 0 and 100.
  2. The system calculates the average of all submissions.
  3. The average is scaled by 0.8 to determine the target value.
  4. The node closest to the target wins and suffers 0 penalty. All other nodes lose 1 stability.
Custom Protocol Modules:
  • Disqualification Override: If 2 or more active nodes choose the exact same value, they are disqualified, suffer -1 penalty, and their numbers are ignored in calculations. (Active at ≤ 4 nodes).
  • Precision Spike: If a node hits the target value precisely, the winner loses 0, but all other nodes suffer double penalty (-2 stability). (Active at ≤ 3 nodes).
  • Zero-One Override: In 1v1 mode, if one node chooses 0 and the other chooses 100, the node choosing 100 overrides all calculations and wins the round. (Active at 2 nodes).
